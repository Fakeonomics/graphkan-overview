Abstract

We present GraphKAN, the first Kolmogorov-Arnold Network (KAN) with discrete ternary weights {−1, 0, +1}, achieving 1.58 bits per parameter. Our 4-phase quantization-aware training pipeline yields a counterintuitive result: quantization to ternary weights improves accuracy from 94.77% (float) to 96.15% (ternary) on MNIST — a regularization-by-quantization effect.

The resulting model occupies 15.4 KB, fitting within the L1 cache of mainstream microcontrollers without requiring floating-point hardware. Random ternary KAN features with analytical least-squares output (ELM mode) achieve 99.3% of full backpropagation accuracy at 31.6 KB.

Novel Contributions

  • ✦ First ternary KAN — All prior KANs use FP32 or 4-bit quantization
  • ✦ Regularization-by-quantization — Accuracy increases stepwise during QAT (94.77% → 96.15%)
  • ✦ Graph topology — Arbitrary directed graph with cyclic message passing
  • ✦ Extreme compression — 15.4 KB, fits Cortex-M0+ L1 cache
