Abstract

We present GraphKAN, the first Kolmogorov-Arnold Network with discrete ternary weights {−1, 0, +1}, achieving 1.58 bits per parameter. Our 4-phase quantization-aware training pipeline yields a counterintuitive result: quantization to ternary weights improves accuracy from 94.77% (float) to 96.15% (ternary) on MNIST — a regularization-by-quantization effect.

The resulting model occupies 15.4 KB, fitting within the L1 cache of mainstream microcontrollers without requiring floating-point hardware. Random ternary KAN features with analytical least-squares output (ELM mode) achieve 99.3% of full backpropagation accuracy at 31.6 KB.
